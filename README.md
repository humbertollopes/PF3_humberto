# ProvaTecnica-Poupança

A partir das entradas abaixo deve-se calcular o valor futuro poupado (VF) 
	- valor aplicado mensalmente (P) 
	- taxa de juros (i) i/100
	- número de parcelas mensais (n)

Fórmula
    - VF = (P*(((1+i)^n-1)/i))

Exemplo 
    - (100*(((1+0,01)^120-1)/0,01)) = 23003,87
